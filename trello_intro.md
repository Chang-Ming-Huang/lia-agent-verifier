# 專案簡介：壽險公會業務員登錄查詢自動化系統

這是一個為了優化行政流程而開發的自動化工具，主要用於快速驗證保險業務員是否符合「年繳方案」的資格（即是否為一年內新進業務）。

系統能自動連線至壽險公會網站查詢，並透過 Trello Webhook 實現「開票即驗證」的自動化流程。

## 核心功能

1.  **全自動查詢**：機器人自動操作瀏覽器，輸入證號並破解圖片驗證碼 (OCR)。
2.  **資格智慧判斷**：自動讀取「初次登錄日期」，判斷是否在一年內，並區分「審核通過」、「資格不符」或「無效證號」。
3.  **Trello 深度整合**：
    *   **自動觸發**：當 Trello 卡片標題包含關鍵字（如「年繳方案申請」）時，自動啟動查詢。
    *   **結果回報**：查詢後的**截圖**會自動上傳至卡片附件。
    *   **Draft 回信**：根據查詢結果（通過/不通過），自動產生對應的 Email 標題與內文，並留言在卡片上，方便客服人員直接複製使用。

## 使用技術

*   **核心語言**: Python (Flask)
*   **瀏覽器自動化**: Playwright (模擬真實使用者行為)
*   **驗證碼識別**: ddddocr (AI OCR 文字辨識)
*   **API 整合**: Trello API & Webhook
*   **部署環境**: Docker 容器化 @ Render
